HPE Nimble Storage Content Collection for Ansible
=================================================

The HPE Nimble Storage Content Collection for Ansible is a set of Ansible modules to manage HPE Nimble Storage arrays. The collection is available on `Ansible Galaxy <https://galaxy.ansible.com/hpe/nimble/>`_ and `Automation Hub <https://cloud.redhat.com/ansible/automation-hub/hpe/nimble>`_.

Source code is available on `GitHub <https://github.com/hpe-storage/nimble-ansible-modules>`_.

   This documentation is a placeholder until the `Collection Index <https://docs.ansible.com/ansible/latest/collections/index.html>`_ has been updated on docs.ansible.com.
