HPE Nimble Storage Content Collection for Ansible¶

The HPE Nimble Storage Content Collection for Ansible is a set of Ansible modules to manage HPE Nimble Storage arrays. The collection is available on Ansible Galaxy and Automation Hub.

Source code is available on GitHub.

This documentation is a placeholder until the Collection Index has been updated on docs.ansible.com.
